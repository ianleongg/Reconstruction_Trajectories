#### 2. Reconstuct trajectory data

* The raw input data is saved in pickled files and can be loaded using a [helper function](./helpers.py)

  - The data list loaded contains four fields, correspond to:
   - timestamp (seconds)
   - displacement (meters)
   - yaw_rate (rads / sec)
   - acceleration (m/s/s)
